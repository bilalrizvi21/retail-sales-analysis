# Retail Sales Analysis Dashboard

An end-to-end Exploratory Data Analysis (EDA) project in Python investigating retail transaction data to uncover sales trends, regional distribution, and product profitability.

## 📊 Dashboard Overview
* **Monthly Sales & Profit Trend:** Tracks seasonal fluctuations and baseline profitability.
* **Product Performance:** Analyzes total sales volume across major categories (Books, Electronics, etc.).
* **Regional Share:** Visualizes distribution across geographic sectors.
* **Cyclical Patterns:** Identifies weekly sales spikes and drop-offs to optimize promotion scheduling.

## 💡 Key Insights
1. **Top Performer:** *Books* drives the highest total sales volume ($370K+), though *Sports* yields the highest individual profit margin (26.38%).
2. **Weekly Cyclicality:** Sales experience a massive spike on Fridays, contrasting sharply with a weekend drop-off.
3. **Data Quality Note:** Identified an "Unknown" segment accounting for minor revenue across categories and regions, highlighting opportunities for future data-cleansing pipelines.

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
